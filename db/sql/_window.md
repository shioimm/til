# ウィンドウ関数
- 集約せずに集約関数を使用する
  - グループ化 - `集約関数 over(partition by グループ化すカラム)`
  - ソート - `集約関数 over(order by ソートキー)`
  - グループ化 + ソート - `集約関数 over(partition by グループ化するカラム order by ソートキー)`

```sql
select
    address
    , count(*) over(partition by address)
from
    customers;

select
    name
    , count(*) over(order by name asc)
from
    customers;
```

#### 使用できる集約関数
- 通常の集約関数
- `rank` / `dense_rank`
- `row_number` ...
