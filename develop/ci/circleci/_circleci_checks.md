# CircleCI Checks
- 参照: [CircleCI Checks の有効化](https://circleci.com/docs/ja/2.0/enable-checks/)
- 参照・引用: CitrcleCI実践入門 第五章

## TL;DR
- GitHubのPRに個別のジョブのステータスを表示する
- GitHubにChecksタブを追加する
  - ビルドステータスを表示する

## Get Started
- Organization Settings -> VCS -> Manage GitHub checks
  - 有効化 / 無効化
  - ブランチプロテクションの設定
