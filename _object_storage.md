# オブジェクトストレージ
- データをオブジェクト単位で扱うストレージのアーキテクチャ
- ファイルを書き込む際にIDが発行され、そのIDを元に読み書きを行う
- 主にオンラインストレージで利用
- ストレージサーバー (ノード) の追加により容量の拡張を無停止で行うことが可能

#### 事例
- Amazon Simple Storage Service
- Google Cloud Storage（英語版）
- Microsoft Azure Blob storage
- Ceph
- OpenStack Swift
