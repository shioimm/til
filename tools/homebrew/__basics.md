# Homebrew
- Mac用パッケージマネージャー
- コマンドラインツールのインストールに使用する
- ex. vim, ruby-build, heroku

#### keg-only
- インストールのみを行い、シンボリックリンクを作成しない
- 必要時は明示的にPATHの設定が必要

#### Homebrew-Cask
- Homebrewの拡張
- GUIアプリケーションのインストールに使用する
- ex. chromedriver, visual-studio-code
