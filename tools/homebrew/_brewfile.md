# Brewfile
- [Homebrew/homebrew-bundle](https://github.com/Homebrew/homebrew-bundle)

```
# 現在の環境からBrewfileを生成
$ brew bundle dump

# インストール
$ brew bundle

# エントリの削除
$ brew bundle cleanup
```

| 構文   | 意味             |
| -      | -                |
| `brew` | `$ brew install` |
| `tap`  | `$ brew tap`     |
| `cask` | `$ brew cask`    |
| `mas`  | `$ brew mas`     |
