# Envoy
- クラウドネイティブ環境のために設計されたL4/L7機能を持つプロキシソフトウェア
- Kubernetesの利用環境では各アプリケーションのサイドカーコンテナとして利用される

## 主な機能
- L3/L4フィルターアーキティクチャ
- HTTP L7フィルターアーキティクチャ
- HTTP/2サポート
- HTTP/3サポート
- HTTP L7ルーティング
- gRPCサポート
- サービスディスカバリー
- ダイナミックコンフィグレーション
- ヘルスチェック
- 負荷分散
- フロント/エッジプロキシサポート
- 統計情報の提供
