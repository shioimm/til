# `od(1)`
- バイナリファイルを (デフォルトでは8進数で) ダンプして出力する (octal dump)

```
$ od <出力フォーマット> ファイル名

# ASCII文字表示を追加
$ od -t x1z -A <出力フォーマット> ファイル名
```

| 型 | 出力フォーマット          |
| -  | -                         |
| a | 文字の名前 (7bit ASCII)   |
| c | ASCII文字かエスケープ文字 |
| d | 符号付き10進数            |
| f | 浮動小数点数              |
| o | 8進数                     |
| u | 符号なし10進数            |
| x | 16進数                    |
