# ドメインの売買
- 一般ユーザーはリセラやレジストラを仲介してレジストリとドメイン名登録契約を結ぶ

## ドメイン売買事業者
### ICANN
- TLD(トップレベルドメイン)を策定する
- TLDを管理するレジストリを承認する
- ルートネームサーバーを管理する

### レジストリ(登録管理組織)
- TLDを管理する
  - 一つのTLDにつき一つのレジストリが管理する
  - `.jp` `.com` `.net` `.org` etc
- ドメイン名をレジストラに卸す

### レジストラ(登録事業者)
- ドメイン名をリセラに卸す
- ドメイン名を一般ユーザーに販売する

### リセラ(再販事業者)
- ドメイン名を一般ユーザーに販売する

## Whois
- ドメインを所有している組織・担当者名・連絡先・有効期限
- レジストリが管理・公開を行う
- 最低一年に一回は更新を行う
