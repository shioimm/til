# モバイルネットワーク
## モバイルネットワークの世代
| 世代 | データ転送速度 | レイテンシ | 解説                                                         |
| -    | -              | -          | -                                                            |
| 1G   | データ転送なし | -          | アナログシステム                                             |
| 2G   | 100-400Kbit/秒 | 300-1000ms | アナログシステムと重複or並行して運用されるデジタルシステム   |
| 3G   | 0.5-5Mbit/秒   | 100-500ms  | アナログシステムと並行して運用される専用デジタルネットワーク |
| 4G   | 1-50Gbit/秒    | < 100ms    | デジタル、パケットのみのネットワーク                         |
