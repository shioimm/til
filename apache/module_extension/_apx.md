# `apxs`
## オプション
| コマンド | 意味                                   |
| -        | -                                      |
| `-c`     | コンパイル                             |
| `-i`     | インストール                           |
| `-a`     | LoadModule設定を生成                   |
| `-n`     | モジュール名を指定                     |
| `-I`     | インクルードするヘッダを探すパスを追加 |
| `-L`     | ライブラリを探すパスを追加             |
| `-l`     | リンクするライブラリを指定             |
| `-Wc`    | 追加でコンパイラに渡すフラグを指定     |
| `-Wl`    | 追加でリンカに渡すフラグを指定         |
