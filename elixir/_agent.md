# エージェント
- プログラミングElixir 第21章

## TL;DR
- 状態を持つバックグラウンドプロセス
- 開始時に渡した関数によって初期状態が決まる

## API
### `Agent.start`
- エージェントを生成する
- 指定の関数に対して、現在のエージェント自身の状態を引数として実行した返り値を
  自身の状態とする

### 引数
- 関数を指定する

### 返り値
- 関数の実行結果の成否とエージェントのPIDのタプル

### `Agent.get`
- エージェントの状態を問い合わせる
- 指定の関数に対して、現在のエージェント自身の状態を引数として実行する

### 引数
- エージェントのPID、関数を指定する

### 返り値
- 渡された関数を呼び出した返り値

### `Agent.update`
- エージェントの状態を更新する
- 指定の関数に対して、現在のエージェント自身の状態を引数として実行した返り値を
  自身の状態とする

### 引数
- エージェントのPID、関数を指定する

### 返り値
- 関数の実行結果の成否
