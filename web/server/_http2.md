# HTTP/2対応
## Webサーバーの接続形態
- 参照: よくわかるHTTP/2の教科書P64-65
- (a)HTTP/2を直接受ける
```
クライアント -> HTTP/2 -> サーバー
```
- (b)HTTP/2をリバースプロキシする
```
クライアント -> HTTP/2 -> プロキシサーバー -> HTTP/2 -> バックエンドサーバー
クライアント -> HTTP/2 -> プロキシサーバー -> HTTP/1 -> バックエンドサーバー
```
- (c)TLSの終端のみ行う
```
クライアント -> HTTP/2 over TLS -> TLS終端サーバー -> HTTP/2 -> バックエンドサーバー
```
