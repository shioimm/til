# 擬似ヘッダ
- 参照: よくわかるHTTP/2の教科書P94-95

## HTTP/2における通信のメタ情報
- HTTP/2において、HTTPヘッダ以外の通信のメタ情報は擬似ヘッダとして扱われ、
  HTTPヘッダと同じようにHEADERSフレームに格納される
- プロキシサーバーがHTTP/2をHTTP/1.1に変換する際、
  擬似ヘッダからリクエストラインやレスポンスラインを生成する

### 擬似リクエストヘッダ
- HTTPメソッド -> :method
- URIスキーム -> :scheme
- URIホスト名 + ポート番号 -> :authority
- URIパス -> :path

### 擬似レスポンスヘッダ
- ステータスコード-> :status
