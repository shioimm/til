## インストール
[Install OCaml](https://ocaml.org/docs/install.html)
- OCamlとOPAMをインストール
  - OPAM -> OCamlのパッケージマネージャ

```ruby
# macOS
brew install ocaml
brew install opam
```

## トップレベル対話環境
- `ocaml`コマンドで実行
  - Ctrl + cで停止

## 演算子など
- ;; -> 入力の区切り
- 整数同士の四則演算は + - * / mod
- 実数同士の四則演算は +. -. \*. /.
- 実数のべき乗は **(べき乗は実数のみに用意されている)
  - 整数と実数の計算を行うと型エラー
- 文字列の連結は ^
- 論理演算は not && ||
  - 優先順位は not > && > ||
- 比較演算は > >= == < <=
  - 整数・実数の区別はない

## 変数
- 変数定義 -> `let 変数名 = 式 ;;`
- 変数の先頭はアルファベット小文字

## 関数
- 関数定義 -> `let 関数名 引数 = 式 ;;`
  - 引数が複数の場合 -> `let 関数 x y ;;`
- `val x : A -> B = <fun>` -> 「float型の値を受け取り、float型の値を返す」
  - 引数が複数の場合 -> `val x : A -> B -> C = <fun>`
    - 右に結合する(一つ目の引数にAを取り、二つ目の引数にBを取り、結果としてCを返す)
