# ar(1)
- アーカイブの作成

```
# スタティックライブラリの場合はアーカイブファイル名のprefixにlibを付与
# アーカイブファイル名、アーカイブファイルに含めるファイル名を指定

$ ar -rcs liblib.a lib.o

# -r - .aファイルが既に存在する場合は書き換える
# -c - フィードバックなしにアーカイブを作成する
# -s - .aファイルの先頭にインデックスを作成する
```

```
# アーカイブファイルの内容を確認

$ ar -t FILE_NAME
```

```
# アーカイブファイルから特定のファイルを取り出す

$ ar -x liblib.a lib.o
```
