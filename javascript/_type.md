# 型
### プリミティブ型 (基本型)
- 真偽値 (Boolean)
- 数値 (Number) - e.g. 42, 3.14159
- 巨大な整数 (BigInt) - e.g. 9007199254740992n (ES2020~)
- 文字列 (String)
- シンボル (Symbol) - e.g. Symbol("sym") (ES2015~)
- undefined
- null

#### falsyな型
- false
- undefined
- null
- 0
- 0n
- NaN
- "" (空文字列)
  
### オブジェクト (複合型)
- プリミティブ型以外のデータ
  - オブジェクト
  - 配列
  - 関数
  - 正規表現
  - Date etc
