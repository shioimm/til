# 関数定義

```js
// 関数宣言
function fn1() {
}

fn1();

// 関数式
const fn2 = function() {
};

fn2();

// 関数式 (Arrow Function)
const fn3 = () => {
};

fn3();
```

### メソッド定義

```js
const obj = {
  method1 () {
  },
  method2: () => {
  }
};

obj.method();
```
