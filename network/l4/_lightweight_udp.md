# UDP-Lite (Lightweight User Datagram Protocol)
- 軽量UDP
- UDPのチェックサム検算の要求条件を下げ、ペイロード中の指定の範囲 (カバー範囲) が壊れていなければ
  アプリケーションに引き渡す
- 音声や動画の伝送を目的に使用される
- データグラムにUDPデータグラム長を表現するフィールドが含まれていないため、
  IPデータグラムの全長からIPヘッダ長 + 8バイトを引いた数をデータグラム長と見做して扱される

## データグラムフォーマット
### 送信元ポート番号 (16ビット)
- データの送信元サービス・オプション

### 送信先ポート番号 (16ビット)
- データの送信先サービス

### カバー範囲 (16ビット)
- ペイロードが壊れていないことを保証する範囲の指定

### チェックサム (16ビット: オプション)
- チェックサム

### ペイロード
- ペイロード
