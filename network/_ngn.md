# NGN (Next Generation Network)
- インターネットと同じIPを基盤とするデータ通信技術を用いて構築された公衆交換電話網
  - e.g. NTT西日本・東日本フレッツ・光ネクスト
- 交換機による回線交換方式を基盤とするアナログ電話回線網を置き換えるもの

#### NGN折り返し通信
- NGNに接続された機器同士でのP2P通信 (IPv6を利用)
