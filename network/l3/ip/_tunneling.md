# IPトンネリング
- 通信プロトコルをカプセル化して他のプロトコルのデータとして通信する技術
- トンネルの手前に該当するノードがヘッダを付加し、トンネルの出口に該当するノードがヘッダを取り除く

#### 事例
- IPv6またはIPv4パケットが自らと異なるIPv4またはIPv6プロトコルのネットワーク上を通過できるよう、
  パケットにIPv4ヘッダまたはIPv6ヘッダを付加する

#### パケットが通過するノード
- ルーター - ルーター
- ホスト - ルーター
- ホスト - ホスト
