# IPv6パケットフォーマット
- バージョン (4ビット)
- トラフィッククラス (8ビット)
  - IPパケットの優先度
  - IPv4におけるToSと同様の機能
- フローラベル (20ビット)
  - 通信フロー (一連のデータのまとまり) を識別する
- ペイロード長 (16ビット)
- ネクストヘッダ (8ビット)
  - IPペイロードを構成するトランスポート層プロトコルの種類を示す
  - IPv4におけるプロトコルフィールドと同様の機能
- ホップリミット (8ビット)
  - ホップ数の上限
- 送信元IPv6アドレス (128ビット)
- 送信先IPv6アドレス (128ビット)
- IPv6ペイロード
