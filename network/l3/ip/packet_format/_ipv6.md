# IPv6パケットフォーマット
- バージョン (4ビット)
- トラフィッククラス (8ビット)
  - サービス種別を表す
  - IPv4におけるToSと同様の機能
- フローラベル (20ビット)
  - 通信フロー (一連のデータのまとまり) を識別するためのラベル
- ペイロード長 (16ビット)
  - ヘッダを除いたIPペイロード長
- ネクストヘッダ (8ビット)
  - IPペイロードを構成するトランスポート層プロトコルの種類を示す番号
  - 追加のオプションヘッダ (拡張ヘッダ) を表す場合もある (経路制御ヘッダ、フラグメントヘッダ、ICMPv6など)
- ホップリミット (8ビット)
  - ホップ数の上限
- 送信元IPv6アドレス (128ビット)
- 送信先IPv6アドレス (128ビット)
- IPv6ペイロード
