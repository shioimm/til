# 近隣キャッシュ
- 近隣ノードのIPv6アドレスとリンク層アドレスの対応
- ノードごとにエントリとして管理される

## エントリのステート
- INCOMPLETE
  - 該当するエントリに対してアドレス解決が進行中
- REACHABLE
  - 該当するエントリに対応する近隣ノードへ到達可能
- STALE
  - 該当するエントリに対応する近隣ノードは到達可能とはいえない
- DELAY
  - 該当するエントリに対応する近隣ノードは到達可能とはいえないものの
    送出済みのパケットがあり、それを待機している
- PROBE
  - 該当するエントリに対応する近隣ノードの到達性を確認する応答を待っている
